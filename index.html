OpenShelf – Online Bibliotheek

Een slimme webapp voor boekenbeheer met Blazor, API & Azure Database

Omschrijving

OpenShelf is een moderne webapp waarmee gebruikers eenvoudig boeken kunnen toevoegen, bekijken en beheren. De applicatie maakt gebruik van een Blazor WebApp, gekoppeld aan een Blazor API en een Azure SQL-database. Via de API kunnen gebruikers 3 van de 4 CRUD-operaties uitvoeren: Create, Read en Delete.

CRUD Operaties in OpenShelf

  • Create: Boeken toevoegen aan de database met titel, auteur, ISBN en coverafbeelding.
  • Read: Boeken bekijken via de webinterface met zoek- en filtermogelijkheden.
  • Delete: Beheerders kunnen boeken veilig verwijderen via een admin-dashboard.
